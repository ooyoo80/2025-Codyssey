# Markdown 주요 문법 정리

## 1. 헤더 (Headers)
- `#` 기호 개수로 제목의 크기를 조절한다. (`#` ~ `######`)

## 2. 링크 (Links)
- `[텍스트](URL)` 형식으로 작성한다.

## 3. 코드 블록 (Code Blocks)
- 백틱(\`\`\`) 3개로 감싸고, 언어를 지정하면 문법 강조가 적용된다.

## 4. 이미지 삽입 (Images)
- `![대체 텍스트](이미지 경로)` 형식으로 작성한다.

## 5. 목록 (Lists)
- 순서 없는 목록: `-`, `*`, `+` 기호 사용
- 순서 있는 목록: `숫자.` 형식 사용
